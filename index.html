Statistical analysis of RDF Linked data

Retrieve data from various publicly accessible SPARQL endpoints as Linked Data, run an analysis on the data and the tool will provide the result of analysis back to you in order to visualise and interpret it.

Analyse Data

Perform SPARQL query

Run SPARQL queries in order to retrieve biological data as Linked data from multiple RDF stores.

Analyse data via OntoFUNC tool

Choose the desired ontology, the accuracy of the analysis and perform one of the for tests (wilcoxon test, hyper test, binominal test, 2X2 conting test)

Visualise and interpret data

After the Linked data will be analysed through the OntoFunc tool the result will be presented in a simplistic format which is both easy to read and interpret.

Download data

The result can be either downloaded in a text file or send as an email.
